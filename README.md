## 如何安裝

1. git clone https://github.com/hrh47/todo-list-v1.git
2. cd todo-list-v1
3. npm install 
4. install mongodb
5. sudo npm install pm2 -g
6. cp .env.example .env
7. add secret key and port number in .env
8. pm2 start index.js
